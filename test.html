Pure CSS Dark Mode

Deze website gebruikt alleen HTML en CSS om een dark mode functionaliteit te implementeren. Dit werkt zonder JavaScript door gebruik te maken van de :has() CSS selector.

Hoe het werkt

De schakelaar is een checkbox input. Met de CSS :has() selector kunnen we de hele webpagina laten reageren op de status van deze checkbox, zonder JavaScript nodig te hebben.

Voordelen

Als JavaScript is uitgeschakeld in de browser, blijft deze functionaliteit gewoon werken! Dit verbetert de toegankelijkheid en gebruikerservaring.
