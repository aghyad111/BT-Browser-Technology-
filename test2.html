JavaScript Dark Mode

Deze website gebruikt JavaScript om een dark mode functionaliteit te implementeren. De schakelaar voegt een class toe aan het body element.

Hoe het werkt

Als de gebruiker op de schakelaar klikt, voegt JavaScript een 'dark-mode' class toe aan het body element. Dit triggert de CSS-stijlen voor de dark mode.

Voordelen

Het gebruiken van JavaScript biedt meer flexibiliteit en maakt het mogelijk om de voorkeur van de gebruiker op te slaan in localStorage voor toekomstige bezoeken.
